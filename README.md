# Telegram Study Bot

Учебный Telegram-бот для изучения Python, Java и Kotlin.  

## Возможности

- Ответы на вопросы по Python, Java, Kotlin
- Распознавание текста с фото
- Распознавание речи из голосовых сообщений
- Анализ длительности видео
- Кнопки для выбора языка и темы
